# /project-mapper:map-init

Create `.claude/` documentation scaffolding with 4 template files. No agents — handle directly. spec.md is managed separately by `/project-mapper:map-spec generate`.

## Steps

### Step 1 — Scaffold

1. Create `.claude/` directory if it does not exist
2. Write 4 template files using the exact templates below:

**map.md**:
```
# Project Map
> Auto-generated by project-mapper

## Entry Points

## Data Flow

## Components

## Structure
```

**design.md**:
```
# Architecture
> Auto-generated by project-mapper

## Stack

## Patterns

## Decisions
```

**backlog.md** (MUST include these sections):
```
# Backlog

## Planned

## In Progress

## Discovered TODOs
```

**changelog.md**:
```
# Changelog

## Recent

## Archive
```

### Step 2 — Report

3. Report: "Init complete — created .claude/ with 4 template files. Run `/project-mapper:map-spec generate` to create spec.md."

## Rules

- Do NOT spawn any Task agents
- If files already exist, overwrite with fresh templates
- Do NOT use AskUserQuestion — init is fully non-interactive
- Do NOT create spec.md — that is managed by the map-spec skill
